# Screenplay Pattern

Key points:
- **Screenplay Pattern** helps your automated tests focus on business workflows and reduce the complexity of your framework
- **Screenplays** are sequences of _activities_ performed by _actors_ in a test scenario
- **Actors** represent _users_ and _external systems_ interacting with the system under test
